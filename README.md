# Terras Altas Turismo

Website conceito para marketplace de turismo regional com 3 páginas principais:

- Descoberta/início
- Listagem de eventos
- Página do evento

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Deploy na Vercel

1. Suba este projeto para um repositório no GitHub.
2. Importe o repositório na Vercel.
3. Use as configurações padrão de Next.js.
4. Clique em Deploy.

O projeto já está estruturado para a Vercel com Next.js App Router.
